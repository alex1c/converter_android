# Руководство по созданию иконок для RuStore

Это руководство поможет вам создать профессиональные иконки для публикации приложения "Универсальный конвертер" в RuStore.

## Требования RuStore

Для публикации в RuStore требуется:
- **Иконка приложения**: 512x512 пикселей, PNG формат, без прозрачности
- **Скриншоты**: минимум 2, максимум 8 скриншотов (рекомендуется 4-6)
- **Промо-изображение**: 1024x500 пикселей (опционально, но рекомендуется)

## Текущая структура иконок

В проекте уже настроены адаптивные иконки (Adaptive Icons) для Android:
- `drawable/ic_launcher_background.xml` - фон иконки с градиентом
- `drawable/ic_launcher_foreground.xml` - передний план с символом конвертации
- `mipmap-anydpi-v26/ic_launcher.xml` - конфигурация адаптивного иконка

## Создание иконки 512x512 для RuStore

### Вариант 1: Использование Android Studio

1. Откройте проект в Android Studio
2. Перейдите в **File > New > Image Asset**
3. Выберите **Launcher Icons (Adaptive and Legacy)**
4. Настройте:
   - **Foreground Layer**: Используйте существующий `ic_launcher_foreground.xml` или загрузите изображение
   - **Background Layer**: Используйте существующий `ic_launcher_background.xml` или выберите цвет
   - **Legacy Icon**: Убедитесь, что включен
5. Нажмите **Next** и **Finish**
6. Экспортируйте иконку 512x512 из созданных ресурсов

### Вариант 2: Использование онлайн-инструментов

#### Рекомендуемые инструменты:
1. **Android Asset Studio** (https://romannurik.github.io/AndroidAssetStudio/)
   - Загрузите ваше изображение или используйте встроенные шаблоны
   - Настройте цвета и стиль
   - Экспортируйте все размеры

2. **App Icon Generator** (https://appicon.co/)
   - Загрузите изображение 1024x1024
   - Автоматически создаст все необходимые размеры

3. **Canva** (https://www.canva.com/)
   - Создайте дизайн 512x512 пикселей
   - Используйте шаблоны для иконок приложений
   - Экспортируйте в PNG

### Вариант 3: Использование Figma/Adobe Illustrator

1. Создайте новый файл 512x512 пикселей
2. Используйте следующие элементы дизайна:
   - **Фон**: Градиент от #6750A4 (фиолетовый) к #4568DC (синий) к #B06AB3 (розово-фиолетовый)
   - **Основной символ**: Две стрелки в противоположных направлениях (символ конвертации)
   - **Дополнительные элементы**: Линии измерения, единицы измерения
3. Экспортируйте в PNG с разрешением 512x512

## Дизайн-рекомендации

### Цветовая схема
- **Основной цвет**: #6750A4 (Primary Light из темы приложения)
- **Акцентный цвет**: #4568DC (синий)
- **Дополнительный цвет**: #B06AB3 (розово-фиолетовый)
- **Текст/Иконки**: #FFFFFF (белый)

### Элементы дизайна
1. **Символ конвертации**: Две стрелки в противоположных направлениях
2. **Единицы измерения**: Маленькие символы (кг, м, л и т.д.)
3. **Градиентный фон**: Создает современный и привлекательный вид
4. **Минимализм**: Простой, но узнаваемый дизайн

### Требования к качеству
- **Разрешение**: Минимум 512x512 пикселей (рекомендуется 1024x1024 для лучшего качества)
- **Формат**: PNG без прозрачности
- **Размер файла**: Оптимизированный (обычно 50-200 KB)
- **Контраст**: Высокий контраст для лучшей видимости на разных фонах

## Создание скриншотов для RuStore

### Требования:
- **Количество**: Минимум 2, максимум 8
- **Формат**: PNG или JPG
- **Разрешение**: Минимум 320px по меньшей стороне
- **Соотношение сторон**: 16:9 или 9:16 (в зависимости от ориентации)

### Рекомендуемые скриншоты:
1. **Главный экран** - Список категорий конвертеров
2. **Экран конвертации** - Пример конвертации веса (кг → фунты)
3. **Экран электричества** - Калькуляторы (закон Ома)
4. **Темная тема** - Показ поддержки темной темы
5. **Разные категории** - Демонстрация разнообразия конвертеров

### Инструменты для создания скриншотов:
1. **Android Studio**: Device Manager → Screenshot
2. **ADB**: `adb shell screencap -p /sdcard/screenshot.png`
3. **Эмулятор**: Встроенная функция скриншота

## Создание промо-изображения (опционально)

### Требования:
- **Размер**: 1024x500 пикселей
- **Формат**: PNG или JPG
- **Содержание**: Название приложения, ключевые функции, визуально привлекательный дизайн

### Рекомендуемый контент:
- Название: "Универсальный конвертер"
- Подзаголовок: "Конвертируйте единицы измерения быстро и точно"
- Иконки категорий: Вес, Длина, Объем, Температура и т.д.
- Градиентный фон в цветах приложения

## Проверка перед публикацией

Перед загрузкой в RuStore убедитесь:
- [ ] Иконка 512x512px в формате PNG
- [ ] Иконка оптимизирована (размер файла < 500 KB)
- [ ] Иконка имеет высокий контраст и читаема в маленьком размере
- [ ] Скриншоты показывают основные функции приложения
- [ ] Все изображения соответствуют требованиям RuStore

## Быстрый старт

Если вы хотите быстро создать иконку 512x512:

1. Откройте `drawable/ic_launcher_foreground.xml` и `drawable/ic_launcher_background.xml`
2. Используйте Android Studio Image Asset Studio для генерации всех размеров
3. Экспортируйте иконку 512x512 из созданных ресурсов
4. Или используйте онлайн-инструмент App Icon Generator с вашим дизайном

---

**Примечание**: Текущие векторные иконки уже настроены и будут работать на всех устройствах Android. Для RuStore вам нужно создать статическую PNG иконку 512x512 на основе этого дизайна.
