# Инструкция по созданию иконки 512x512 для RuStore

## Быстрый способ через Android Studio

1. **Откройте Android Studio** и ваш проект
2. Перейдите в **File → New → Image Asset**
3. В открывшемся окне:
   - **Icon Type**: Выберите "Launcher Icons (Adaptive and Legacy)"
   - **Foreground Layer**:
     - **Source Asset**: Выберите "Image" или "Clip Art"
     - Если используете изображение: загрузите файл с символом конвертации (две стрелки)
     - Если используете Clip Art: выберите подходящую иконку
     - **Scaling**: 100%
     - **Shape**: None (или Circle для круглой иконки)
   - **Background Layer**:
     - **Color**: #6750A4 (основной цвет приложения)
     - Или используйте градиент, создав его в отдельном файле
   - **Legacy Icon**: Убедитесь, что включен
4. Нажмите **Next** → **Finish**
5. После генерации найдите файл `ic_launcher.png` в папке `mipmap-xxxhdpi` (это будет 192x192)
6. Для получения 512x512:
   - Откройте этот файл в графическом редакторе
   - Увеличьте до 512x512 с сохранением качества
   - Или используйте онлайн-инструмент для масштабирования

## Альтернативный способ: Использование онлайн-инструментов

### Вариант 1: App Icon Generator (https://appicon.co/)
1. Создайте изображение 1024x1024 пикселей с вашим дизайном
2. Загрузите на appicon.co
3. Скачайте готовую иконку 512x512

### Вариант 2: Android Asset Studio (https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html)
1. Загрузите ваше изображение или используйте встроенные шаблоны
2. Настройте цвета и стиль
3. Скачайте все размеры, включая 512x512

## Рекомендуемый дизайн иконки

### Основные элементы:
- **Фон**: Градиент от #6750A4 (фиолетовый) через #4568DC (синий) к #B06AB3 (розово-фиолетовый)
- **Основной символ**: Две стрелки в противоположных направлениях (↔) - символ конвертации
- **Цвет символа**: Белый (#FFFFFF) для контраста
- **Стиль**: Минималистичный, современный

### Размеры элементов (для 512x512):
- **Стрелки**: ~200px в высоту, ~150px в ширину каждая
- **Центральный круг**: ~80px в диаметре
- **Отступы**: Минимум 50px от краев

## Проверка качества

Перед загрузкой в RuStore убедитесь:
- ✅ Размер точно 512x512 пикселей
- ✅ Формат PNG
- ✅ Нет прозрачности (сплошной фон)
- ✅ Высокое качество (не размыто)
- ✅ Хорошо читается в маленьком размере (проверьте на 48x48)
- ✅ Размер файла оптимизирован (< 500 KB)

## Готовые шаблоны

Вы можете использовать следующие инструменты для создания иконки на основе текущего дизайна:

1. **Figma**: Создайте 512x512, используйте градиент и векторные стрелки
2. **Adobe Illustrator**: Профессиональный инструмент для векторной графики
3. **Canva**: Простой онлайн-редактор с шаблонами иконок
4. **GIMP**: Бесплатный графический редактор

## Пример дизайна

```
┌─────────────────────────────────┐
│  [Градиентный фон]              │
│                                 │
│      ←  ○  →                    │
│     (стрелки и круг)            │
│                                 │
│  [Декоративные линии]           │
└─────────────────────────────────┘
```

**Цвета градиента** (сверху вниз, слева направо):
- Начало: #6750A4 (фиолетовый)
- Середина: #4568DC (синий)
- Конец: #B06AB3 (розово-фиолетовый)

---

**Важно**: После создания иконки 512x512, сохраните её в безопасном месте. Она понадобится при загрузке приложения в RuStore.
